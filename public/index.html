API Endpoints

Users
Path Method Description
/users/signup POST Add/register a new user
/users/login POST Checks credentials and logs in the user.
/users GET Retrieves a list of all users.
/users/:id GET Retrieves details of a specific user by ID.
/users/:id DELETE Removes a user account by ID.
/users/:id PATCH Updates details of a specific user by ID.
/users/profile GET Get Retrieves details of a specific user by ID.
/users/updatepassword PATCH Updates password of a specific user
Products
Path Method Description
/products POST Add a new product to the catalog.
/products GET Retrieves a list of all products.
/products/:id GET Retrieves details of a specific product by ID.
/products/:id DELETE Delete a product from the catalog by ID.
/products/:id PATCH Updates details of a specific product by ID.
Cart
Path Method Description
/cart POST Add a product to the user's cart.
/cart GET Retrieves a list of all products in the cart.
/cart/:userID GET Retrieves a list of all products in a users cart.
/products/:userID/:prodID DELETE Delete a product from the user's cart
/products/clearAll DELETE Delete all products from the user's cart
/products/:id PATCH Updates quantity of a specific product in the cart.
WishList
Path Method Description
/wishlist POST Add a product to the user's wishlist.
/wishlist GET Retrieves a list of all wishlist in the wishlist.
/wishlist/:userID GET Retrieves a list of all wishlist in a user's wishlist.
/wishlist/:userID/:prodID DELETE Delete a product from the user's wishlist
/wishlist/clearAll DELETE Delete all products from the user's wishlist
/wishlist/:id PATCH Updates quantity of a specific product in the wishlist.
